
SUMMARY
-------

Admin Toggle allows site admins to toggle the display of admin items on a page
with the press of a single key. This allows admins to easily preview what their
site looks like to non-admins.


REQUIREMENTS
------------

None.


INSTALLATION
------------

Go to admin/modules and enable the Admin Toggle module.


CONFIGURATION
-------------

Admin Toggle can be configured via Configuration » Administration » Admin
Toggle. User permissions can be configured via Administration » People »
Permissions.
